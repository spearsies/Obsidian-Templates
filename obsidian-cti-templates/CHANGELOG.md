## [1.0.0] - 2025-11-05

### Added

#### Core Templates
- **APT Group Profile Template** - Comprehensive threat actor documentation
- **Campaign Tracking Template** - Campaign analysis and attribution
- **CTI Threat Actor Profile** - Formal intelligence report format
- **Intelligence Requirements** - Framework for prioritizing research
- **IOC Hash Template** - File hash indicator tracking with sandbox analysis
- **IOC IP Template** - IP address tracking with geolocation and reputation
- **IP Tracking Template** - Simplified IP documentation
- **Malware Analysis Template** - Technical malware documentation
- **Campaign Template** - Streamlined threat actor campaign tracking
- **URL Tracker** - Malicious URL documentation
- **GitHub Link Template** - Security repository tracking
